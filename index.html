moveByManualAsync(vx_max, vy_max, z_min, duration, drivetrain=0, yaw_mode=<instance> { 'is_rate': True, 'yaw_or_rate': 0.0}, vehicle_name='')¶

Read current RC state and use it to control the vehicles.

Parameters sets up the constraints on velocity and minimum altitude while flying. If RC state is detected to violate these constraints then that RC state would be ignored.

Parameters:
  • vx_max (TYPE) – max velocity allowed in x direction
  • vy_max (TYPE) – max velocity allowed in y direction
  • vz_max – max velocity allowed in z direction
  • z_min (TYPE) – min z allowed for vehicle position
  • duration (TYPE) – after this duration vehicle would switch back to non-manual mode
  • drivetrain (TYPE, optional) – when ForwardOnly, vehicle rotates itself so that its front is always facing the direction of travel. If MaxDegreeOfFreedom then it doesn’t do that (crab-like movement)
  • yaw_mode (TYPE, optional) – Specifies if vehicle should face at given angle (is_rate=False) or should be rotating around its axis at given rate (is_rate=True)
